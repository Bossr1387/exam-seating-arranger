# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-10-27

### Added
- Initial release of Exam Seating Arranger
- Intelligent round-robin seating arrangement algorithm
- Flexible column configuration system
- Support for tab-separated and CSV data formats
- File upload functionality (CSV, TSV, TXT)
- Real-time data preview
- Configurable minimum spacing between same organization
- Advanced options (header detection, column preservation, randomization)
- Results validation tool
- Export options (copy to clipboard, download CSV)
- Multilingual data support (Chinese, Japanese, Arabic, etc.)
- Fully responsive design (mobile, tablet, desktop)
- Accessible interface with ARIA labels
- Local storage for configuration persistence
- Example data loader (English and Chinese samples)
- Comprehensive statistics display
- Print-friendly results layout
- Cross-browser compatibility (Chrome, Firefox, Safari, Edge)

### Features
- **No Dependencies**: Pure HTML/CSS/JavaScript
- **Privacy-First**: All processing happens locally in browser
- **Professional UI**: Modern gradient design with smooth animations
- **User-Friendly**: Intuitive interface with helpful tooltips
- **Flexible**: Adaptable to various exam scenarios and data formats

### Documentation
- Comprehensive README with usage instructions
- Contributing guidelines for developers
- MIT License
- Example data files (English and Chinese)
- Inline code documentation
